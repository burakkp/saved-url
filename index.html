SavedURL Chrome Extension

Description

This extension allows you to save webpage links so that you can view them later. It only uses your computer's local memory and does not connect to an external server or save any personal information.

Features

  • Save your URLs to view later
  • Uses only local computer memory
  • No external server connection
  • Protects your personal information

Installation Guide

  1. Visit the Chrome Web Store.
  2. Search for "SavedURL" extension.
  3. Click "Add to Chrome" to install.
